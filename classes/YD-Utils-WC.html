WooCommerce Utility Class

This class provides a set of utility methods for interacting with WooCommerce features, such as checking WooCommerce support, retrieving address fields, formatting prices, handling product reviews, and converting exceptions to WordPress errors. It includes both static methods for WooCommerce-related functionalities and helper functions that are commonly used throughout the WooCommerce integration.
